# 摩旅助手 App

摩旅助手 - 摩托车旅行助手应用

## 技术栈

- UniApp + Vue 3 + TypeScript
- Pinia 状态管理
- Vite 构建

## 项目结构

```
src/
├── pages/          # 页面
├── components/     # 组件
├── store/          # 状态管理
├── services/      # API 服务
├── utils/          # 工具函数
├── hooks/          # 组合式函数
├── constants/     # 常量
└── types/          # 类型定义
```

## 运行

```bash
# 安装依赖
npm install

# 开发模式
npm run dev:h5

# 构建
npm run build:h5
```
